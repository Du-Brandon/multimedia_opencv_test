# 使用說明
## 用pip安裝必要模組
```
pip install opencv-python
pip install mediapipe
```
- **若無法安裝則請確認python版本是否為3.9~3.12之間**

## 執行check_version確認版本
```
OpenCV version:  4.11.0
MediaPipe version:  0.10.21
NumPy version:  1.26.4
PIL version:  11.1.0
```

## 說明
### 直接執行 main.py

- **預設為無慮鏡**
- **按B會出現灰階綠鏡**
- **按C會出現彩色綠鏡**
- **按S會出現素描綠鏡**
- **按N會恢復無綠鏡模式**
- **按Q會出現退出程式**
